JCS_TweenPathAction

The point to point simple path action that uses tween to move.

Variables

NameDescription
PointsList of points for setting up the path.
mRandomRandom Random the path by randomizing the target point
ContinueTweenDo continue tween instead of just tween.

Require Components

  • JCS_TransformTweener
  • JCS_AdjustTimeTrigger
