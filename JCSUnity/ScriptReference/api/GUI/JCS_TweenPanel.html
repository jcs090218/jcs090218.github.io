JCS_TweenPanel

Panel that can do tween effect.

Variables

NameDescription
OverrideTweenOverride the tween animation while is still playing.
mActiveSoundSound played when active this panel.
mDeactiveSoundSound plays when this panel is deactive.

Functions

NameDescription
ActiveTween to the taget position and play sound effect.
DeactiveTween back to the starting position and play sound effect.

Require Components

  • JCS_TransformTweener
  • JCS_SoundPlayer
